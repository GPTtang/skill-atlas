## Workflow

1. Show theme options to the user (describe or list the 10 themes above)
2. Ask which theme to apply — wait for explicit selection
3. Apply the selected theme's colors and fonts consistently throughout the artifact
4. Ensure proper contrast and readability across all elements

## Application Rules

When applying a theme:
- Use **Primary** for headings, key elements, and borders
- Use **Accent** for highlights, callouts, and interactive elements
- Use **Background** for slide/page backgrounds
- Use **Text** for body copy
- Apply **Header font** to H1, H2, titles, slide headers
- Apply **Body font** to paragraphs, bullet points, captions
- Maintain consistent contrast ratio (minimum 4.5:1 for text)

## Custom Theme Creation

If none of the 10 themes fit the user's needs:

1. Ask for: color preferences, mood/tone, brand colors if any
2. Generate a new theme following the same structure (Primary / Accent / Background / Text / Fonts)
3. Give it a descriptive name
4. Show for review before applying

## Example

**Input**: "Apply a theme to my slide deck — something professional and tech-focused."

**Output**:
- Suggest "Tech Innovation" or "Modern Minimalist"
- Wait for selection
- Apply: electric teal headings, dark background (or light mode), Exo 2 headers, Roboto body
- Ensure all text maintains readability against the background
